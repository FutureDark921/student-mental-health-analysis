

## Student Social Media & Mental Health Analysis

## Project Overview
This project analyzes the relationship between social media usage and student mental health using Google Sheets. The analysis focuses on understanding how factors such as daily social media usage, sleep habits, physical activity, academic level, and platform preference may influence mental well-being.
The project follows a complete data analytics workflow, including data cleaning, feature engineering, exploratory data analysis, dashboard development, and business recommendations.

## Objectives
- Analyze the impact of social media usage on mental health.
- Identify platforms associated with higher stress levels.
- Examine the relationship between sleep quality and mental well-being.
- Compare mental health scores across academic levels and genders.
- Build an interactive dashboard for data-driven insights.

## Tools Used
- Google Sheets
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
- Data Visualization

## Dataset Information
- Dataset: Student Social Media and Mental Health Impact
- Records: 5,000 Students
- Features: Demographic, Academic, Lifestyle, and Mental Health Variables

## Key Fields
- Age
- Gender
- Country
- Academic Level
- Most Used Platform
- Purpose of Use
- Avg Daily Usage Hours
- Daily Unlocks
- Study Hours
- Physical Activity Hours
- Sleep Hours Per Night
- Stress Level
- Mental Health Score

## Data Cleaning Process
The following data preparation steps were performed:
- Checked for duplicate records
- Validated missing values
- Standardized text formatting
- Verified numerical ranges
- Created analytical categories
- Organized data for pivot table analysis

- Moderate (5–7)
- Good (> 7)

## Key Insights
- Students with higher social media usage generally reported lower mental health scores.
- Healthy sleep habits were associated with improved mental well-being.
- Stress levels varied across social media platforms.
- Academic level and lifestyle factors influenced mental health outcomes.
- Physical activity showed a positive relationship with mental health scores.

## Recommendations
- 1.Encourage balanced social media usage among students.
- 2.Promote healthy sleep habits to improve mental well-being.
- 3.Support programs that encourage physical activity.
- 4.Provide targeted interventions for high-stress student groups.
- 5.Monitor platform-specific behavioral trends.
